URL Shortener Microservice

Please write url with [http/https]

Example how to create

https://3-url-shortener-fcc-ivan.glitch.me/new_url/https://stackoverflow.com

Example of outputs

For correct url:

{
  "original_url": "https://stackoverflow.com",
  "short_url": "https://3-url-shortener-fcc-ivan.glitch.me/blbu1o"
}

For incorrect url:

{
  "error": "Wrong url format"
}

Example of usage

Put https://3-url-shortener-fcc-ivan.glitch.me/blbu1o in browser and it will be redirect to https://stackoverflow.com

For non existing short url:

{
  "error": "There is no this url in db"
}
